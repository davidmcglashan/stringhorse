cat [string]

Adds [string] to the end of each line.

flatten

Flattens the original text's lines into one long string.

flattens

Flattens the original text's lines into one long space-separated string.

grep [string]

Retains only lines which contain [string].

head [number]

Keep only the first [number] characters on each line.

info

Shows some info about the output: lines, words, chars, etc.

lower

Converts the original text into lower case.

ngrep [string]

Retains only lines which do not contain [string].

precat [string]

Adds [string] to the start of each line.

r [a] ([b])

Replaces the first instance of [a] on each line with [b]. If [b] is not specified then [a] is replaced with '' (i.e. it is removed).

rafter [string]

Removes the text after first instance of [string] on each line, leaving [string] in place.

rall [a] ([b])

Replaces every instance of [a] on each line with [b]. If [b] is not specified then all the [a]s are replaced with '' (i.e. they are removed).

rbefore [string]

Removes the text before first instance of [string] on each line, leaving [string] in place.

reverse

Reverses the order of the original text.

rhead [number]

Remove the first [number] characters on each line.

rtail [number]

Remove the last [number] characters on each line.

sort

Sorts the original text alphabetically.

sortza

Sorts the original text reverse-alphabetically.

tail [number]

Keep only the last [number] characters on each line.

tokenise

trim

Removes the whitespace from the start and end of each line.

trimblanks

Removes any empty lines from the original text.

upper

Converts the original text into upper case.

wwwify

Pretends the original text is a list of hyperlinks and displays them as anchor tags so they can be clicked and visited.
