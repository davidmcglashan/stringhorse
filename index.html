cat [string]

Adds [string] to the end of each line.

See also precat

clines (space|tab)

Collapses the original text's lines into one long string, optionally separated by a tab or a space.

column ([number])

Removes everything except column [number], where a column is a whitspace-separated string. [number] defaults to 1.

See also rcolumn

cspaces (tab)

Collapses any runs of spaces in the original text's lines into one single space or TAB.

See also ctabs

ctabs (space)

Collapses any runs of TABs in the original text's lines into one single TAB or space.

See also cspaces

decode

Decode each line as if it's an encoded URL e.g. %20 becomes a space, and so on.

See also encode

encode

Encode each line for safe use in a URL e.g. spaces become %20, and so on.

See also decode

explode

Explode the text by inserting blank lines in between the existing ones.

See also split

grep [string]

Retains only lines which contain [string].

See also ngrep

head [number]

Keep only the first [number] characters on each line.

See also rhead | rtail | tail

info

Shows some info about the output: lines, words, chars, etc.

kalphas (tab|space)

Keeps only the alphabet characters in each line. Optional second parameter defines whether to separate these with TABs or spaces.

See also knumbers | ksymbols

knumbers (tab|space)

Keeps only the numbers in each line. Optional second parameter defines whether to separate these with TABs or spaces.

See also kalphas | ksymbols

ksymbols (tab|space)

Keeps only the symbols in each line. Optional second parameter defines whether to separate these with TABs or spaces.

See also kalphas | knumbers

lower

Converts the original text into lower case.

See also upper

ngrep [string]

Retains only lines which do not contain [string].

See also grep

nsort ([number])

Sorts the original text's lines naturally (e.g. 1, 2, 10). [number] can be provided to sort by a column.

See also reverse | sort

number ([number])

Number the lines starting at 1, or at [number] if it is provided.

precat [string]

Adds [string] to the start of each line.

See also cat

r [a] ([b])

Replaces the first instance of [a] on each line with [b]. If [b] is not specified then [a] is replaced with '' (i.e. it is removed).

See also rafter | rall | rbefore

rafter [string]

Removes the text after first instance of [string] on each line, leaving [string] in place.

See also r | rall | rbefore

rall [a] ([b])

Replaces every instance of [a] on each line with [b]. If [b] is not specified then all the [a]s are replaced with '' (i.e. they are removed).

See also r | rafter | rbefore

ralphas

Removes the alphabet characters from each line, leaving only whitespace, the symbols, and the numbers.

See also rnumbers | rsymbols

rbefore [string]

Removes the text before first instance of [string] on each line, leaving [string] in place.

See also r | rafter | rall

rcolumn ([number])

Removes column [number], where a column is a whitspace-separated string. [number] defaults to 1.

See also rcolumn

reverse

Reverses the order of the original text.

See also sort | nsort

rhead [number]

Remove the first [number] characters on each line.

See also head | rtail | tail

rnumbers

Removes the numbers from each line, leaving only whitespace, the symbols, and the alphabet characters.

See also ralphas | rsymbols

rsymbols

Removes the symbol characters from each line, leaving only whitespace, the numbers, and the alphabet characters.

See also ralphas | rnumbers

rtail [number]

Remove the last [number] characters on each line.

See also head | rhead | tail

sort ([number])

Sorts the original text's lines alphabetically. [number] can be provided to sort by a column.

See also nsort | reverse

split ([string])

Split the lines on occurances of [string] (or a space if not provided), removing the [string] in the process.

See also explode

tail [number]

Keep only the last [number] characters on each line.

See also head | rhead | rtail

trim

Removes the whitespace from the start and end of each line.

See also trimblanks

trimblanks

Removes any empty lines from the original text.

See also trim

twords ([number])

Splits the original text into words separated by TABs and reformats the text with [number] words on each line, separated by TABs. [number] defaults to 1 when omitted.

See also words

upper

Converts the original text into upper case.

See also lower

words ([number])

Splits the original text into words separated by whitespace and reformats the text with [number] words on each line. [number] defaults to 1 when omitted.

See also twords

wwwify

Pretends the original text is a list of hyperlinks and displays them as anchor tags so they can be clicked and visited.
